- `UPI expense tracker` (Targeting Indian users searching for UPI-specific tracking)
- `offline budget planner` (Capturing users looking for zero-cloud/private apps)

### 2. Secondary Keywords (Broad & Feature-Specific)

- `no SMS expense manager` (Capturing users repelled by apps requiring broad SMS permissions)
- `offline transaction log` (Technical audit search intent)
- `Lakh formatting ledger` (Specific formatting requirement)

| "automatic offline budget app India" | Tech-savvy Indian user wanting automation but zero server sync. | Group keywords around "offline-first automatic" and "GPay/Paytm notification listener". |
| "UPI budget ledger app offline" | Financial planners looking for ledger books or digital cash books. | Optimize for "ledger", "cash book", and "bahi khata" terms in long description metadata. |

### 4. Competitor Differentiation & Placement
Our core competitors (e.g., Walnut, SMS Organizer, Fold, Spendee) rely heavily on reading the user's SMS inbox, linking bank accounts via SMS/AA, or syncing to cloud servers. UPI Unified Expense Tracker stands apart by offering:
- **Zero Network Dependency**: Completely runs in airplane mode.
- **Zero Dangerous Permissions**: Only requests `NotificationListenerService` rather than broad READ_SMS permissions which are heavily restricted by Google Play.
- **Privacy Pledge**: Open-source transparency, completely client-side local database encryption.
